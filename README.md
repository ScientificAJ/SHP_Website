# Student HackPad 2026 - Idea2Impact

A cosmic adventure where imagination meets innovation. This platform challenges students worldwide to design tools that improve the student experience.

## Deployment

This site is deployed to GitHub Pages.

URL: [https://scientificaj.github.io/SHP_Website/](https://scientificaj.github.io/SHP_Website/)

## Features

- **Project Gallery**: Explore outstanding submissions.
- **Rules & Guidelines**: Information on how to participate.
- **Updates**: Latest news and announcements.
- **Discussions**: Community hub for collaboration.
- **Authentication**: Simple local-storage based auth for sign-up and login.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
